free-gjc-starter · 왕초보 풀 가이드

컴퓨터에 무료 AI 코딩 비서(gjc) 깔기. 터미널 여는 법부터 → 프로그램 설치 → 무료 키 발급 → 설정까지 전부.

순서
0. 어떤 방식으로 쓸지 고르기 1. 준비물 2. 터미널 여는 법 3. gjc 설치 4. 이 무료 세팅 설치 5. 무료 API 키 발급 6. 키를 컴퓨터에 넣기 7. 메인 두뇌 정하기 (A/B) 8. 확인 & 첫 대화 문제 해결
⚠ 딱 하나만 지키세요 — 발급받은 키는 비밀번호입니다. 남에게 보여주거나 파일·채팅에 붙여넣지 말고, 아래 안내대로 본인 PC의 환경변수에만 넣으세요. 이 세팅 파일에는 키가 이름으로만 들어갑니다.

0. 어떤 방식으로 쓸지 고르기

메인 두뇌(가장 똑똑한 역할)를 뭘로 할지 두 갈래입니다. 나머지 일꾼들은 두 경우 다 무료입니다.

방식 A Gemini를 메인으로 추천

구글 계정으로 로그인만 하면 됩니다. 카드 등록 X.
  • Google AI 구독 있음 → 그대로 사용
  • 구독 없음 → Gemini CLI 무료 티어로도 됩니다
필요한 무료 키: OpenRouter, Groq (2개)
방식 B Gemini 없이, NVIDIA GLM-5.2를 메인으로

구글/Gemini를 아예 안 쓰고 싶을 때. NVIDIA의 무료 GLM-5.2(753B)를 메인으로 씁니다.
  • 로그인 대신 NVIDIA 무료 키 사용
필요한 무료 키: OpenRouter, Groq, NVIDIA (3개)

※ 방식 A로 해도 Gemini가 잠깐 막히면 자동으로 GLM-5.2로 넘어가도록 이미 설정돼 있습니다(Gemini 1순위 → GLM 2순위). 그래서 A여도 NVIDIA 키를 넣어두면 더 안정적입니다.

1. 준비물 (5분)

  • 구글 계정 하나 (Gmail 있으면 됨) — 무료 키 사이트 로그인·Gemini 로그인에 씀
  • 인터넷 되는 윈도우 PC 또는 맥
  • 돈 안 듭니다.

2. "터미널" 여는 법

터미널 = 컴퓨터에 글자로 명령을 내리는 창입니다. 여기다 아래 명령들을 복사-붙여넣기합니다.

🪟 윈도우
  1. 키보드 ⊞ 윈도우 키를 누릅니다.
  2. 터미널 이라고 칩니다. (안 뜨면 powershell)
  3. Windows Terminal 또는 Windows PowerShell 클릭 → 창이 뜹니다.
🍎 맥
  1. ⌘ + 스페이스 를 동시에 → 검색창.
  2. 터미널 또는 terminal 입력.
  3. 터미널(Terminal) 클릭.
복사-붙여넣기 & 실행 — 아래 상자 명령을 복사(Ctrl+C, 맥 ⌘+C) → 터미널에 붙여넣기(윈도우: 마우스 오른쪽 클릭, 맥: ⌘+V) → Enter 를 눌러야 실행됩니다.

3. gjc 설치

🪟 윈도우 (PowerShell)
irm bun.sh/install.ps1 | iex

👉 끝나면 창을 닫고 터미널을 새로 여세요. 그다음:

bun install -g gajae-code
🍎 맥 / 리눅스
curl -fsSL https://bun.sh/install | bash

👉 끝나면 터미널을 새로 여세요. 그다음:

bun install -g gajae-code
확인: gjc --version 을 치고 버전 숫자가 나오면 성공.
❔ command not found / 인식할 수 없습니다 → 터미널을 완전히 닫고 다시 여세요.

5. 무료 API 키 발급

일꾼 AI가 쓸 무료 열쇠를 받습니다. 카드 등록 필요 없어요. 각 사이트에서 구글 로그인 → Create Key → 이름 입력 → 만들기 → 키 복사.

사이트발급 링크키 형태언제 필요?
OpenRouteropenrouter.ai/keyssk-or-v1-...항상 (A·B 공통)
Groqconsole.groq.com/keysgsk_...항상 (A·B 공통)
NVIDIA NIMbuild.nvidia.com → Get API Keynvapi-...방식 B 필수 / 방식 A는 선택(안정성↑)
키는 생성 직후 한 번만 보이는 경우가 많습니다. 바로 복사해서 다음 단계로 가세요.

6. 키를 컴퓨터에 넣기

여기에_키 부분만 방금 복사한 진짜 키로 바꿔서 붙여넣으세요. (안 쓰는 키 줄은 빼도 됩니다)

🪟 윈도우 (PowerShell)
setx OPENROUTER_API_KEY "여기에_키"
setx GROQ_API_KEY "여기에_키"
setx NVIDIA_API_KEY "여기에_키"

👉 그리고 터미널을 닫고 새로 여세요. (setx는 새 창부터 적용)

🍎 맥 / 리눅스
echo 'export OPENROUTER_API_KEY="여기에_키"' >> ~/.zshrc
echo 'export GROQ_API_KEY="여기에_키"'       >> ~/.zshrc
echo 'export NVIDIA_API_KEY="여기에_키"'     >> ~/.zshrc
source ~/.zshrc

(오래된 셸이면 ~/.zshrc → ~/.bashrc 로 바꿔 쓰세요.)

7. 메인 두뇌 정하기

방식 A Gemini를 메인으로 (로그인만)

터미널에 free-gjc 입력 → gjc가 뜨면 아래 중 하나를 입력하고 Enter → 브라우저에서 본인 구글 계정 로그인.
상황입력할 명령
Google AI 구독 있음/login google-antigravity
구독 없음(무료로)/login google-gemini-cli
무료(google-gemini-cli)로 쓸 거면 설정 파일의 메인 모델 한 줄만 바꿉니다.
파일: ~/.gjc-free/agent/config.yml (윈도우는 %USERPROFILE%\.gjc-free\agent\config.yml) — 메모장/텍스트편집기로 열어서: 
modelRoles:
  default: google-gemini-cli/gemini-2.5-pro   # 이 줄만 이렇게

방식 B Gemini 없이 NVIDIA GLM-5.2를 메인으로

로그인 필요 없습니다. NVIDIA 키(5번에서 발급)만 있으면 됩니다. 설정 파일의 메인 모델 줄을 GLM으로 바꾸세요.
파일 ~/.gjc-free/agent/config.yml 을 열어 default: 줄을: 
modelRoles:
  default: nvidia-nim/z-ai/glm-5.2   # 메인 = 무료 GLM-5.2
폴백 순서(Gemini→GLM)는 이미 들어있으니 default 한 줄만 바꾸면 끝입니다.

8. 확인 & 첫 대화

free-gjc config get modelRoles       # 역할별 모델이 찍히면 설정 OK
free-gjc -p --no-session "안녕? 넌 누구야?"
AI가 대답하면 성공! 🎉 이제 이렇게 씁니다: 
free-gjc                              # 대화형으로 켜기
free-gjc "파이썬으로 계산기 만들어줘"    # 한 번에 시키기
평소 쓰던 gjc(다른 프로필)와 완전히 따로 돌아가니 안심하세요.

문제 해결

증상해결
command not found / 인식할 수 없습니다터미널을 완전히 닫고 새로 열기. 안 되면 3번 설치 재실행.
git ... 없음4번 아래 안내대로 Git 먼저 설치.
No API key / 401 / unauthorized그 provider 키를 6번대로 다시 넣고 터미널 새로 열기. 키 오타·잘못 복사 확인.
no match / 모델 못 찾음무료 모델 이름이 바뀐 것. 유효한 id를 찾아 ~/.gjc-free/agent/ 의 models.yml·config.yml 수정.
OpenRouter 무료 목록 · Groq 모델 · NVIDIA 모델
Gemini 로그인 창이 안 열림터미널에 표시된 주소를 복사해 브라우저 주소창에 직접 붙여넣기.
안심하세요 — 이 repo·이 페이지 어디에도 API 키·개인정보가 없습니다. 키는 당신 PC에만 저장됩니다. 남과 공유할 땐 이 repo 주소만 주면 되고, 기존 gjc 설정은 건드리지 않습니다(~/.gjc-free에만 설치).
