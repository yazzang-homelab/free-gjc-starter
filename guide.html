free-gjc-starter · 무료 API 키 가이드

초보자용. 메인 두뇌는 Gemini(로그인만, 키 불필요), 서브에이전트는 무료 API 키 2개(OpenRouter · Groq)면 끝.

⚠ 보안 원칙
발급받은 키는 절대 이 repo나 어떤 파일에도 붙여넣지 마세요. 오직 본인 PC의 환경변수에만 넣습니다. 이 세팅의 models.yml은 키의 이름(apiKeyEnv)만 참조하므로, 파일을 공유해도 키는 새지 않습니다.

한눈에 — 뭐가 필요한가

역할모델필요한 것비용

→ 발급받을 무료 키는 딱 2개: OPENROUTER_API_KEY, GROQ_API_KEY. Gemini는 로그인만 하면 됩니다.

① OpenRouter 키 발급 무료

  1. openrouter.ai 접속 → 구글/깃허브 계정으로 Sign in.
  2. openrouter.ai/keys 로 이동 → Create Key 클릭.
  3. 이름 아무거나(예: free-gjc) 입력 후 생성 → sk-or-v1-... 키 복사.
  4. 이 세팅은 :free 모델만 쓰므로 결제 등록 없이 무료로 동작합니다. (일부 무료 모델은 일일 한도가 있음)

② Groq 키 발급 무료

  1. console.groq.com 접속 → 구글 계정으로 로그인.
  2. console.groq.com/keys → Create API Key 클릭.
  3. 이름 입력 후 생성 → gsk_... 키 복사. (생성 직후 한 번만 보이니 바로 복사)
  4. Groq는 무료 티어로 매우 빠른 추론 속도를 제공합니다.

③ 환경변수에 키 넣기

macOS / Linux

export OPENROUTER_API_KEY="sk-or-v1-복사한키"
export GROQ_API_KEY="gsk_복사한키"

# 매번 자동 적용하려면 셸 설정파일에 추가:
echo 'export OPENROUTER_API_KEY="sk-or-v1-복사한키"' >> ~/.bashrc   # zsh면 ~/.zshrc
echo 'export GROQ_API_KEY="gsk_복사한키"'          >> ~/.bashrc

Windows (PowerShell)

setx OPENROUTER_API_KEY "sk-or-v1-복사한키"
setx GROQ_API_KEY "gsk_복사한키"
# setx 는 "새로" 여는 터미널부터 적용됩니다 → 창을 닫고 다시 여세요.

④ 메인 Gemini 로그인 키 불필요

free-gjc 를 실행한 뒤 아래 중 하나를 입력해 본인 구글 계정으로 로그인합니다.

상황명령
Google AI 구독이 있음/login google-antigravity
무료로 쓰고 싶음/login google-gemini-cli

무료(google-gemini-cli)로 쓸 땐 ~/.gjc-free/agent/config.yml 의 default 한 줄을 google-gemini-cli/gemini-2.5-pro 로 바꾸세요.

⑤ 확인

free-gjc config get modelRoles      # 역할별 모델이 찍히면 세팅 OK
free-gjc -p --no-session "누구니?"   # 실제 응답이 오면 키/로그인 OK
모델 이름이 안 먹힐 때 — 무료 모델 슬러그는 자주 바뀝니다. 유효한 id를 확인해 ~/.gjc-free/agent/ 의 models.yml/config.yml 에 바꿔 넣으세요.
· OpenRouter 무료 목록: openrouter.ai/models?q=free
· Groq 모델 목록: console.groq.com/docs/models
