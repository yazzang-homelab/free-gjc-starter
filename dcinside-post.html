💻 무료로 AI 코딩 비서(gjc) 깔기 — 왕초보 통합 가이드

메인 두뇌는 Gemini(로그인만, 카드 X), 일꾼들은 무료 AI. 터미널 여는 법부터 설정까지 전부.

⚠ 딱 하나 — 발급받은 키는 비밀번호입니다. 남에게 보여주거나 파일·댓글에 붙여넣지 말고 본인 PC 환경변수에만 넣으세요.

0. 어떤 방식으로 쓸지

방식메인 두뇌필요한 무료 키
A (추천)Gemini — 구독 or 무료 로그인OpenRouter, Groq (2개)
BGemini 없이 NVIDIA GLM-5.2OpenRouter, Groq, NVIDIA (3개)

※ A로 해도 Gemini가 막히면 자동으로 GLM-5.2로 넘어가게 이미 설정돼 있음.

1. 준비물

구글 계정 하나(Gmail), 인터넷 되는 윈도우/맥. 돈 안 듬.

2. 터미널 여는 법

🪟 윈도우 — ⊞윈도우키 → 터미널 입력(안 뜨면 powershell) → 클릭.
🍎 맥 — ⌘+스페이스 → 터미널 입력 → 클릭.

붙여넣기: 윈도우는 마우스 오른쪽 클릭, 맥은 ⌘+V → 그리고 Enter 눌러야 실행.

3. gjc 설치

🪟 윈도우 (PowerShell):

irm bun.sh/install.ps1 | iex

👉 끝나면 터미널 새로 열고:

bun install -g gajae-code

🍎 맥 / 리눅스:

curl -fsSL https://bun.sh/install | bash

👉 터미널 새로 열고:

bun install -g gajae-code
확인: gjc --version 쳐서 버전 숫자 나오면 성공.
❔ command not found → 터미널 완전히 닫고 다시 열기.

4. 무료 세팅 설치

🪟 윈도우:

git clone https://github.com/yazzang-homelab/free-gjc-starter.git
cd free-gjc-starter
.\install.ps1

🍎 맥 / 리눅스:

git clone https://github.com/yazzang-homelab/free-gjc-starter.git
cd free-gjc-starter
bash install.sh

5. 무료 API 키 발급

각 사이트에서 구글 로그인 → Create Key → 이름 입력 → 키 복사.

사이트주소언제
OpenRouteropenrouter.ai/keys항상
Groqconsole.groq.com/keys항상
NVIDIA NIMbuild.nvidia.com/z-ai/glm-5.2 → Get API KeyB 필수 / A 선택

6. 키 넣기

🪟 윈도우 (안 쓰는 줄은 빼도 됨):

setx OPENROUTER_API_KEY "여기에_키"
setx GROQ_API_KEY "여기에_키"
setx NVIDIA_API_KEY "여기에_키"

👉 터미널 새로 열기(setx는 새 창부터 적용).

🍎 맥 / 리눅스:

echo 'export OPENROUTER_API_KEY="여기에_키"' >> ~/.zshrc
echo 'export GROQ_API_KEY="여기에_키"'       >> ~/.zshrc
echo 'export NVIDIA_API_KEY="여기에_키"'     >> ~/.zshrc
source ~/.zshrc

7. 메인 두뇌 정하기

[방식 A] Gemini(무료) — free-gjc 실행 후 /login google-gemini-cli. 기본값이 이미 google-gemini-cli/gemini-2.5-pro라 로그인만 하면 됨.

⚠️ 로그인 전: 브라우저에 구글 계정 여러 개면 엉뚱한 회사 계정으로 인증돼 GOOGLE_CLOUD_PROJECT 오류 남. 시크릿 창에서 개인 지메일 1개만 로그인한 채로 진행하셈.

⚠️ google-antigravity는 쓰지 마셈 — 별도 접근권/구독 없으면 404(Requested entity was not found) 남. Opus 등 Anthropic 모델도 Antigravity 전용이라 동일. (Antigravity 있는 사람만 antigravity 로그인+default 교체)

[방식 B] GLM-5.2 — 로그인 없이 NVIDIA 키만. ~/.gjc-free/agent/config.yml 의 default를:

modelRoles:
  default: nvidia-nim/z-ai/glm-5.2
